# Microservices

This section documents each service in the ForesightX ecosystem. Each page covers purpose, responsibilities, tech stack, API surface, dependencies, and deployment notes.

Services documented here (each links to a dedicated page):

- `ForesightX` (core API/service)
- `ForesightX-auth` (authentication + tokens)
- `ForesightX-data` (data ingestion and normalization)
- `ForesightX-orchestration` (workflows, jobs)
- `ForesightX-Pattern` (data science / model service — deep dive)
- `ForesightX-profile` (user/profile management)
- `foresightx-trading-hub` (trading UI/backend)
- `ForesightX-frontend` (client SPA)

Each page includes practical notes for maintainers and deployers. Where available, links to service READMEs and code are provided.
