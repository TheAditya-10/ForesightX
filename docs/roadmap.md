# Roadmap

The next stage should strengthen measurement and operations before adding product breadth.

## Near term

- Complete automated test, image-build, and deployment workflows.
- Add structured logs, request correlation IDs, metrics, and service dashboards.
- Add TLS, managed secrets, backup verification, and dependency vulnerability scanning.
- Establish repeatable database migration and rollback procedures.

## Model operations

- Monitor input drift, prediction distribution, latency, and post-event accuracy.
- Automate retraining only after evaluation gates and artifact approval.
- Add backtesting and benchmark reports for every promoted model.
- Expand explainability beyond confidence into feature and evidence summaries.

## Platform growth

- Add more instruments, providers, exchanges, and configurable analysis horizons.
- Introduce richer portfolio risk and personalized policy controls.
- Move from single-host Compose to ECS/Fargate or EKS when operational demand justifies it.
- Add high availability, rolling deployment, and tested disaster recovery.
